## ⚙️ Como Executar o Projeto

Para visualizar o projeto localmente, siga os passos abaixo:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/bruno-mariano-silva-portfolio.git
   ```

2. **Navegue até o diretório do projeto:**
   ```bash
   cd bruno-mariano-silva-portfolio
   ```

3. **Abra o arquivo `index.html` no seu navegador de preferência.**

   Não é necessário instalar nenhuma dependência ou executar um servidor local, pois o projeto é composto por arquivos estáticos.
